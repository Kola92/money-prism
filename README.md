# Money Prism - Crypto Currency Tracker

## Description

Money Prism is a simple web application that allows you to track the ever changing value of crypto currencies in real time. It uses the [CoinMarketCap API](https://coinmarketcap.com/api/) to get the latest data. It is built using [React](https://reactjs.org/), [Redux Toolkit Query](https://redux-toolkit.js.org/tutorials/rtk-query) and [Ant Design](https://ant.design/).
